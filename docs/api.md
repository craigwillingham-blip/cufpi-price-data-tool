# API

Base URL: `http://127.0.0.1:8000`

## Endpoints
- `GET /stores`
- `GET /products?query=`
- `GET /products/{id}/prices`
- `POST /receipts/upload`
- `GET /receipts/{id}`
- `POST /crowd/submit`
- `GET /sources`
- `PATCH /sources`
- `POST /ingestion/circulars/run`
- `POST /ingestion/circulars/from-url`
- `POST /ingestion/instacart/scan`
- `POST /ocr/run`
