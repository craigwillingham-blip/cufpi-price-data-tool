# UX

## Home
- Primary search bar and top tasks as large tiles.
- Rationale: supports the most common action without training.

## Compare
- One table with store, size, price, and compare action.
- Rationale: single scanable layout, minimal filters.

## Receipt Upload
- Dropzone + instant item list for correction.
- Rationale: reduces uncertainty and makes errors visible.

## Contribute
- Three fields only, advanced options hidden.
- Rationale: crowdsourcing requires speed.

## Sources
- Inline editable list with enable toggle.
- Rationale: configuration should be safe and clear.
